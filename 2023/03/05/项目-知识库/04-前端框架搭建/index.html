第四天

Vue Cli

vue cli = vue + 一堆组件
文档

创建vue cli项目

  1. 安装依赖

    首先需要安装node.js,然后使用npm命令下载vue cli依赖

  2. 创建web项目

    选择 Manually select features

  3. 启动

前端项目结构

  • index.html(首页), main.ts(初始启动配置文件，地位相当于pom.xml), APP.vue(初始内容页面)

    vue cli初始执行main.ts, 将内容页APP.vue渲染到index.html,完成页面显示
    集成第三方组件，一般在main.ts里加配置

src目录下

  • assert:放静态资源,public目录下也会放静态资源，但assert会参与打包，public不会参与打包
  • components:组件
  • router: 路由
  • store: 存放全局变量
  • views: 初始页面内容页面

index.html 引入外部静态文件(public目录下的静态资源)用<%=BASE_URL %>

Vue cli 需要编译才能发布,最后将dist包上传服务器
